Nextjs Astro Micro is Astro Micro Theme for Next.js. It's a fork of trevortylerlee's popular theme `Astro Micro`. Also Astro Micro is fork of Mark Horn's theme `Astro Nano`.

Nextjs Astro Micro is support [MDX](https://nextjs.org/docs/pages/building-your-application/configuring/mdx). however, [Pagefind](https://pagefind.app) module is not supported. but you can integration with static build like [this](https://www.petemillspaugh.com/nextjs-search-with-pagefind)

Micro adds features like [Pagefind](https://pagefind.app) for search, [Giscus](https://giscus.app) for comments, and more. For a full list of changes, see this [blog post](/blog/00-micro-changelog).

Micro still comes with everything great about Nano â€” full type safety, a sitemap, an RSS feed, and Markdown + MDX support. Styled with TailwindCSS and preconfigured with system, light, and dark themes.

Visit [Astro Nano on Github](https://github.com/markhorn-dev/astro-nano) to get started.

Visit [Astro Micro on Github](https://github.com/trevortylerlee/astro-micro) to get started.

Visit [Nextjs Astro Micro on Github](https://github.com/luasenvy/nextjs-astro-micro) to get started.
